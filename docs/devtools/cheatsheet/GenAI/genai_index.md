

# AI

* AI = Artificial Intelligence
  - mimic human behavior 
* ML = Machine Learning
  - statistical methods
* DL = Deep Learning
  - neural networks
  - ANN = Artificial Neural Network
    - CNN = Convolutional Neural Networks (for image classification)
    - RNN = Recurrent Neural Networks (for text classification/ manupilation)
  - NLP = Natural Language Programming
    - LLM (Large Language Models)

* Generative AI = Subset of AI that focuses on creating artificial agents that can generate text, images, music etc

- ML is part of AI
- DL is part of ML
- GenAI is part of DL
